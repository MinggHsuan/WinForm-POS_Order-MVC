# POS點餐系統

## 📖 專案描述
- 這是一個點餐系統，使用MVC的架構及資料流，菜單內容及促銷打折活動，統一用JSON來制定規格。
- 做到任何菜單及其他折扣活動，在一行程式碼不更改且編譯按鈕不能執行的情況下，可以任意抽換餐點菜單及促銷打折功能。
- 最後點餐完成，除了原本的打折活動自選使用外，也可以透過AI推薦功能，依照購物車內容，自動選擇最多折扣直接做使用並回饋選擇理由。

## 🚀 主要功能
- 餐點分類(e.g.主餐、附餐、甜點等)
- 訂餐流程(新增、刪除、更改數量)
- 結帳計算(自動加總金額、折扣活動選擇)
- AI勾選推薦折扣功能

## 🏗️ 技術架構
- **語言**：C#
- **UI框架**：WinForm
- **框架**：.NET Framework
- **架構模式**：MVP
